P31: 4. Metric M1 — Domain Recall
P34: Definition: For each actual amendment change in B7981027 Amendment 4, was the corresponding USDM entity predicted at High or Medium confidence? A prediction counts as a match if the predicted usdmEntity matches the actual entity — regardless of whether the specific change text matches.
P37: M1 Scoring Logic
P39: 5. Metric M2 — Category Accuracy
P42: Definition: For each predicted amendment, does the predicted category match the actual category of the corresponding real amendment? Scored at the amendment level (not change level).
P46: 6. Metric M3 — Hallucination Rate
P49: Definition: A hallucinated prediction is any predicted change where entity_rate = 0 AND training_evidence.count = 0 in the lineage block. This means the generator predicted something with absolutely no benchmark or historical support.
P53: M3 Detection Code
P55: 7. Metric M4 — Lineage Completeness
P58: Definition: Every predicted change must have a fully populated lineage block with all required fields non-null and training_evidence referencing only non-verify-set study IDs.
P62: 8. Metric M5 — Confidence Calibration
P65: Definition: For predictions marked High confidence, what % actually match a real change in B7981027 Amendment 4? This is the most important trust metric — if the generator calls something High confidence, it should be right most of the time.
